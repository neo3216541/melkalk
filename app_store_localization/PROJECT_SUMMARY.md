# MEL Calculator - Project Summary

## Application Overview

MEL Calculator is a professional Flutter application designed for aviation professionals to calculate UTC expiration dates for ICAO Minimum Equipment List (MEL) categories.

## Core Features

### 1. MEL Calculations
- Real-time UTC date calculations for Categories A, B, C, D
- Displays exact expiration dates with timestamps
- Shows remaining days for each category
- Automatic recalculation on settings change

### 2. Customization
- Adjustable delay values for each category
- Category A: Default 3 days
- Category B: Default 10 days
- Category C: Default 30 days
- Category D: Default 120 days
- Save functionality with confirmation

### 3. User Interface
- Clean Material Design 3 interface
- Light, dark, and system theme options
- 12-hour and 24-hour time format support
- Smooth animations and transitions
- Intuitive settings drawer

### 4. Localization
- 30 languages supported
- Full UI localization
- Month names localized
- Language selection dropdown
- Automatic system language detection

### 5. Data Management
- Local storage using SharedPreferences
- No account or registration required
- All data stored on device
- No cloud sync
- Privacy-focused design

## Technical Stack

- Framework: Flutter 3.0+
- Language: Dart
- State Management: Flutter BLoC

## What is MEL

The Minimum Equipment List (MEL) is an ICAO regulatory framework that defines which aircraft equipment can be inoperative while still allowing safe operations.

### MEL Categories
- Category A: Shortest delay (typically 3 days)
- Category B: Short delay (typically 10 days)
- Category C: Medium delay (typically 30 days)
- Category D: Longest delay (typically 120 days)
