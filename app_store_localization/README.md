# App Store Localization Files

## Overview

This directory contains App Store metadata for MEL Calculator in 11 languages. Each file includes promotional text, description, what's new, and keywords optimized for App Store submission.

## File Structure

Each localization file contains:

### 1. Promotional Text (170 characters max)
Short, compelling text that appears at the top of your App Store listing. Can be updated without submitting a new version.

### 2. Description (3,794 characters max)
Full app description that explains features, benefits, and use cases. First 2-3 lines are critical as they appear before "more" button.

### 3. What's New (4,000 characters max)
Version update notes that describe new features, improvements, and bug fixes.

### 4. Keywords (100 characters max, space-separated)
Search terms that help users discover your app. No commas, just spaces between words.

## Character Limits Summary

- Promotional Text: 170 characters
- Description: 3,794 characters
- What's New: 4,000 characters
- Keywords: 100 characters (space-separated)

## App Store Connect Setup

### Step 1: Primary Language
1. Log in to App Store Connect
2. Select your app
3. Go to App Information
4. Set Primary Language to English (US)
5. Copy content from en-US.md

### Step 2: Add Localizations
1. Click "+" to add a new localization
2. Select language from the list
3. Copy content from corresponding .md file
4. Paste into appropriate fields
5. Save changes

### Step 3: Screenshots
For each language, you need:
- iPhone 6.7" Display: 1290 x 2796 pixels (minimum 3, maximum 10)
- iPad 12.9" Display: 2048 x 2732 pixels (minimum 3, maximum 10)

## Usage Instructions

### For App Store Submission

1. Open the .md file for your target language
2. Copy the Promotional Text section
3. Paste into App Store Connect > Promotional Text field
4. Copy the Description section
5. Paste into App Store Connect > Description field
6. Copy the What's New section
7. Paste into App Store Connect > What's New in This Version field
8. Copy the Keywords section
9. Paste into App Store Connect > Keywords field
10. Save and submit for review

### For Updates

When releasing a new version:
1. Update the "What's New" section in all language files
2. Keep Promotional Text and Description unless major changes
3. Review Keywords quarterly and update based on performance
4. Maintain consistency across all languages

## Best Practices

### Promotional Text
- Lead with main benefit
- Include key feature
- Mention language support
- Keep under 170 characters

### Description
- Start with strong opening line
- Use clear section headers
- List key features first
- Explain what MEL is
- Include technical details
- End with call-to-action

### What's New
- List new features first
- Mention improvements
- Include bug fixes
- Keep it concise
- Thank users

### Keywords
- Use singular forms (Apple indexes plurals)
- No spaces after separators
- Include brand terms
- Add category terms
- Use relevant aviation terms
